Find and recover your missing post types.

== Description ==

Did you lost some posts after install new WP Theme?
This plugin may help you to find them.
Install and just click; it's easy.

== Installation ==

= Installation =
1. Download the plugin zip package and extract it.
1. Put the folder named "where-are-my-post-types" under /wp-content/plugins/ directory
1. Go to the plugins page in your Wordpress admin panel and click "Activate"
